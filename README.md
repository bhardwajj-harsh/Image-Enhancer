# 🖼️ Image Enhancer

A simple and efficient image enhancement application that improves image quality using various processing techniques. This project allows users to enhance images by adjusting brightness, contrast, and applying filters.

---

## 🚀 Features

* Image brightness and contrast adjustment
* Noise reduction
* Sharpening and smoothing filters
* Easy-to-use interface
* Real-time image processing

---

## 🛠️ Tech Stack

* Python
* OpenCV
* NumPy
* Flask (if web-based) / Tkinter (if GUI-based)

---

## 📂 Project Structure

```
Image-Enhancer/
│── app.py
│── requirements.txt
│── static/
│── templates/
│── images/
```

---

## ▶️ How to Run

```bash
git clone https://github.com/bhardwajj-harsh/Image-Enhancer.git
cd Image-Enhancer
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```
